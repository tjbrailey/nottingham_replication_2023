----------------------------------------------
		READ ME
----------------------------------------------

This repository contains the data and scripts necessary for reproducing the results in the article "Does Competence Make Citizens Tolerate Undemocratic Behavior?" by Kristian Vrede Skaaning Frederiksen, published in the American Political Science Review.

The repository contains the following files:

QUESTIONNAIRES:
		Quest_main_US.pdf
		Quest_main_UK.pdf
		Quest_main_CZ.pdf    (before translation)
		Quest_main_Kor.pdf   (before translation)
		Quest_main_Mex.pdf  (before translation)


DATA FILES:
		dataset.dta  (final dataset for analysis)


STATA AND R SCRIPTS:
		Data_prep.do  (transforms raw data to final dataset)
		Analysis.do      (produces all results as presented in the main article and Dataverse appendix)
		Power.R           (provides power calculations for Appendix H)

APPENDIX:
                                    Dataverse_Appendix.pdf (The long Dataverse appendix containing various auxiliary analyses)
